# School App MVP

A lightweight student information system designed to manage academic records, student profiles, and basic school analytics.

## Purpose

This project explores how simple digital systems can improve school data management and student tracking in real educational environments.

## Features

- Student profile management
- Academic record tracking
- Basic dashboard analytics
- Lightweight data storage system

## Tech Stack

- Python
- Streamlit
- SQLite / CSV

## Structure

- app.py → main application
- database.py → data storage logic
- pages/ → UI modules
- models.py → data structures
- utils.py → helper functions

## Status

MVP Phase (early development)
